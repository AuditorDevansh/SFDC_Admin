⚡ Quick Cheat Sheet
The most important facts, limits, and distinctions — all in one place for fast revision.
🔐 Security Distinctions
Profile Controls WHAT you can do (CRUD)
Role Controls WHAT you can see (records)
Permission Set Adds permissions on top of profile
OWD Baseline access — set most restrictive
Sharing Rule Opens access beyond hierarchy
🗄️ Relationship Types
Master-Detail Tight — child deleted with parent
Lookup Loose — child exists independently
Many-to-Many Via junction object
Hierarchical User object only
Roll-Up Summary Master-Detail only
📊 Report Types
Tabular Simple list, no grouping
Summary Group by rows
Matrix Group by rows AND columns
Joined Up to 5 blocks in one report
⚙️ Automation Tools
Flow Builder Current standard — use this
Process Builder Being retired → migrate to Flow
Workflow Rules Being retired → migrate to Flow
Apex Triggers Complex code-based logic
📦 Sandbox Types
Developer No data, refresh 1 day
Developer Pro More storage, refresh 1 day
Partial Copy Sample data, refresh 5 days
Full All data, refresh 29 days
⚠️ Key Limits
MD Relationships Max 2 per object
Lookup Relationships Max 40 per object
Custom Fields Up to 500 per object
Dashboard Components Max 20 per dashboard
Cross-Object Formula Max 10 levels deep
💼 Sales Cloud Flow
Lead Unqualified prospect
Convert Lead Creates Acct + Contact + Opp
Opportunity Potential deal with stages
Account Company / organisation
Contact Person at an account
🎧 Service Cloud Essentials
Case Core support request object
Queue Holds unassigned records
Assignment Rule Auto-routes cases
Escalation Rule Escalates unresolved cases
Entitlement Defines SLA for customer
